ValidationService

Servicio encargado de validar parámetros y asegurar la existencia del directorio temporal en la aplicación.

Métodos

Método Descripción
validateSegmentSize(int segmentSizeKB) Valida que el tamaño del segmento esté entre 16 KB y 1 GB.
validateFileSize(long fileSize) Valida que el tamaño del archivo no exceda 1 GB.
ensureTempDirectoryExists(String dirPath) Asegura que el directorio temporal exista, creándolo de forma segura en entornos multihilo.

Detalles de los Métodos

validateSegmentSize(int segmentSizeKB)

Verifica que el tamaño del segmento esté dentro de los límites permitidos. Si el tamaño es inferior a 16 KB o superior a 1 GB, o es igual a 0, lanza una IllegalArgumentException con un mensaje descriptivo.

validateFileSize(long fileSize)

Comprueba que el tamaño del archivo no supere 1 GB. Si el archivo excede este límite, lanza una IllegalArgumentException indicando que el archivo es demasiado grande.

ensureTempDirectoryExists(String dirPath)

Verifica si el directorio especificado existe. Si no existe, lo crea de forma segura utilizando sincronización para entornos multihilo. Retorna el objeto File que representa el directorio.
