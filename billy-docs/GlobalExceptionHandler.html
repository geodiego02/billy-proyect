GlobalExceptionHandler

Clase que gestiona de forma centralizada las excepciones dentro de la aplicación.

Métodos

Método Descripción
handleIOException(IOException ex) Maneja errores de entrada/salida (I/O) y devuelve un mensaje genérico.
handleMessagingException(MessagingException ex) Maneja errores de mensajería y envío de correos, retornando un mensaje genérico.
handleMailSendException(MailSendException ex) Maneja errores específicos del envío de correos.
handleMissingServletRequestPartException(MissingServletRequestPartException ex) Maneja errores cuando falta una parte requerida en la solicitud.
handleGeneralException(Exception ex) Maneja excepciones generales y desconocidas.

Detalles de los Métodos

handleIOException(IOException ex)

Maneja errores relacionados con la lectura y escritura de archivos. Registra el error y devuelve una respuesta con estado 500.

  • Registra el error completo en los logs.
  • Devuelve un mensaje genérico para el usuario.

handleMessagingException(MessagingException ex)

Maneja errores en la mensajería y el envío de correos electrónicos. Registra el error y retorna una respuesta con estado 500.

  • El error se registra en detalle en los logs.
  • Se envía un mensaje genérico al usuario para no exponer detalles técnicos.

handleMailSendException(MailSendException ex)

Maneja errores específicos del proceso de envío de correos. Registra el error y devuelve un mensaje informando que ocurrió un error al enviar el correo.

  • Utilizado para errores de envío de correos electrónicos.
  • Devuelve un mensaje genérico sin detalles técnicos.

handleMissingServletRequestPartException(MissingServletRequestPartException ex)

Maneja errores cuando falta una parte requerida en la solicitud multipart/form-data.

  • Registra una advertencia en los logs.
  • Devuelve una respuesta con estado 400.

handleGeneralException(Exception ex)

Maneja cualquier otra excepción inesperada. Registra el error y devuelve una respuesta genérica con estado 500.

  • Captura excepciones no previstas.
  • Devuelve un mensaje genérico para el usuario.
