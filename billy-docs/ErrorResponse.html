ErrorResponse

Clase que representa la respuesta de error personalizada utilizada para devolver mensajes de error en la API.

Atributos

Atributo Tipo Descripción
status int Código de estado HTTP del error.
error String Clave o identificador del error.
message String Mensaje descriptivo del error.

Métodos

  • ErrorResponse(int status, String error, String message): Constructor que inicializa los atributos de la respuesta de error.
  • getStatus(): Retorna el código de estado HTTP del error.
  • getError(): Retorna la clave o identificador del error.
  • getMessage(): Retorna el mensaje descriptivo del error.
