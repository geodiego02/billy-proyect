FileSplitService

Clase encargada de la segmentación de archivos en partes más pequeñas.

Métodos

Método Descripción
splitFile(...) Divide un archivo en segmentos más pequeños y envía actualizaciones de progreso mediante WebSocket.
listSegments(...) Lista los segmentos generados de un archivo.
extractNumericPart(...) Extrae la parte numérica de los nombres de archivos para ordenarlos correctamente.

Detalles de los Métodos

splitFile(File localFile, String originalFilename, int segmentSizeKB, String sessionId)

Divide el archivo en múltiples segmentos basándose en el tamaño especificado y envía actualizaciones de progreso a través de WebSocket.

  • Verifica que el archivo exista y no esté vacío.
  • Elimina segmentos antiguos del mismo archivo.
  • Divide el archivo utilizando un buffer de 8KB.
  • Envía actualizaciones intermedias de progreso y, al finalizar, envía "DONE" si la segmentación se completó.

listSegments(String originalName)

Obtiene una lista ordenada de los nombres de los segmentos generados para un archivo.

  • Filtra y ordena los archivos que comienzan con el nombre original.
  • Ordena numéricamente según la parte numérica del nombre del archivo.

extractNumericPart(String fileName)

Extrae el número de parte del nombre de un archivo segmentado para ordenarlo correctamente.

  • Obtiene la parte numérica después del último punto del nombre del archivo.
  • Convierte el valor a entero; si no es válido, devuelve 0.
