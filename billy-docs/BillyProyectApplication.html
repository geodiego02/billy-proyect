BillyProyectApplication

Clase principal de la aplicación Spring Boot que inicia la ejecución del sistema.

Métodos

Método Descripción
main(String[] args) Inicia la aplicación y configura la carpeta de logs.
crearCarpetaLogs() Verifica si la carpeta de logs existe y, si no, la crea en la raíz del proyecto.

Detalles de los Métodos

main(String[] args)

  • Llama a crearCarpetaLogs() para asegurar que el directorio de logs exista.
  • Inicializa la aplicación con SpringApplication.run(...).
  • Registra un mensaje en los logs indicando que la aplicación ha iniciado.

crearCarpetaLogs()

  • Verifica si la carpeta logs existe en la raíz.
  • Si no existe, la crea con mkdirs().
  • Imprime mensajes de confirmación o error según el resultado.
