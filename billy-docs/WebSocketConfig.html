WebSocketConfig

Clase de configuración para habilitar WebSockets en la aplicación, permitiendo la comunicación en tiempo real.

Métodos

Método Descripción
configureMessageBroker(MessageBrokerRegistry config) Configura el message broker para el envío y recepción de mensajes.
registerStompEndpoints(StompEndpointRegistry registry) Registra los endpoints de WebSocket usando STOMP y habilita SockJS.

Detalles de los Métodos

configureMessageBroker(MessageBrokerRegistry config)

Configura el message broker de WebSockets. Se habilita un broker simple con el prefijo /topic para los mensajes enviados desde el servidor y se establece el prefijo /app para los destinos de mensajes desde el cliente.

  • Habilita el broker simple en /topic.
  • Establece /app como prefijo para mensajes dirigidos al servidor.

registerStompEndpoints(StompEndpointRegistry registry)

Registra el endpoint STOMP para que los clientes puedan conectarse al servidor de WebSocket.

  • Define el endpoint /ws-split para las conexiones WebSocket.
  • Permite conexiones desde cualquier origen mediante setAllowedOriginPatterns("*").
  • Habilita SockJS para clientes que no soporten WebSocket nativo.
